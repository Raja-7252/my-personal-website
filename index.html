Emotion Detection in Faces

A deep neural network model was built to detect emotions in faces using a dataset of grayscale images. Through experimentation with hyperparameters, a validation accuracy of at least 60% was achieved. The model's performance was evaluated using the test dataset, and the Confusion Matrix and Classification Report were generated to assess its effectiveness.The technologies used in this project included Python, Deep Learning, Convolutional Neural Networks (CNN), and Tensor Board.
